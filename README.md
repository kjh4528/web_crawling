# MCP_class 실습 저장소

MCP 서비스 개발자 과정 실습 및 미니 프로젝트 기록용 저장소입니다.

## 작업 폴더

모든 작업은 아래 폴더 기준으로 진행합니다.

C:\Users\UserK\work

## 가상환경

이 저장소는 프로젝트 폴더 내부의 .venv 가상환경을 사용합니다.

## 가상환경 생성
python -m venv .venv

## 가상환경 활성화

PowerShell 기준:
.venv\Scripts\activate

## 파이썬 버전 확인
python --version

## 라이브러리 설치
개인 노트북에서 처음 이어서 작업할 때:
pip install -r requirements.txt
